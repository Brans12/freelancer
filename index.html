Sitio web

El sitio web es una colección de páginas web enlazadas y comunes en un dominio de internet y con una finalidad concreta. Esta se identifica por una dirección URL única, ya que es la raíz del sitio web.

Usualmente las personas suelen usar el termino de sitio web y pagina web de forma indistinta, pero en realidad la definición entre página web y sitio web son completamente diferentes, aunque cabe decir que existen sitios web de una sola página conocidos en inglés como landing pages.

Paginas web

Las páginas web son documentos realizados en HTML, este documento es capaz de contener texto, sonido, video, programas, enlaces, imágenes entre muchas otras cosas.

Tipos de páginas web.

Páginas web estáticas. En estas páginas no se puede cambiar el contenido. La información es permanente, y el usuario de dicha página no puede interactuar la página.

Páginas web dinámicas: Los usuarios puede interactuar con la página para crear y colaborar con el contenido. En estos se puede destacar los blogs, foros y tiendas online.

Tiendas online: Son paginas orientadas a la compra y venta de objetos desde cualquier parte del mundo.
